# 🤖 AMASHIA MD BOT V1.0.0

**Premium WhatsApp Bot | Rapide, Stab, ak Pwisan**

---

## 📖 Deskrisyon

AMASHIA MD se yon bot WhatsApp ki itilize Baileys library. Bot la ofri:

✨ **Karakteristik prensipal:**
- 🎧 Télécharger de la musique
- 📹 Télécharger TikTok
- 🌍 Traduction de texte
- 📥 Sauvegarde automatique de statuts
- 🛡️ Anti-suppression de message

---

## 📦 Instalasyon Lokalman

1. **Klon depo a:**
```bash
git clone https://github.com/TOPFEROS-TECH/AMASHIA-MD-BOT-V-100.git
cd AMASHIA-MD-BOT-V-100
```

2. **Enstale dependencies yo**
```bash
npm install
```

3. **Kreye fichye `.env` si li manke:**
```bash
cp .env.example .env
```

---

## 🚀 Ekzekisyon

1. **Pou devlopman (Hot Reload):**
```bash
npm run dev
```

2. **Produksyon:**
```
npm start
```

---

## 🌐 Tès Web API Lokal:

1. **Verifye koneksyon:**
   - Navigatè URL: `http://localhost:3000`
2. **Verify status bot la**
   ```bash
   curl http://localhost:3000/status
   ```
